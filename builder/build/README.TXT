CombatLogX has different installation steps than other plugins,
please read this guide before trying to add the plugin to your server!

Installation Guide:
1) Make sure your server is completely stopped.
2) Extract the 'CombatLogX.jar' file from the zip into the '/plugins/' folder of your server.
3) Extract the 'SirBlobmanCore.jar' file from the zip into the '/plugins/' folder of your server.
4) Create a new folder called '/plugins/CombatLogX/expansions/' on your server.
5) Extract the expansion jar files into your '/plugins/CombatLogX/expansions/' folder of your server.
6) Start your server so that the configuration files can be generated.
7) Stop your server and then edit the configuration files with the settings that you wish to use.
8) While your server is stopped, you can also remove any expansion jar files that you will not be using.
9) Start your server and enjoy using CombatLogX!
